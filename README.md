Launch
Clone repository

git clone https://github.com/Dm1tr11-Potapov/H2Sunbelt.git
cd H2Sunbelt
Create virtual environment:

python -m venv venv
Launch venv:

.\venv\bin\activate.bat
Install packages:

pip install -r requirements.txt
Launch server:

cd H2Sunbelt
python manage.py runserver
